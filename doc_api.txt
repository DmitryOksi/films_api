node cli.js --task=1 --title='aaa' --release_year=2020 --format='ccc' --stars='ddd'

node cli.js --task=2 --id=602964b9737a927b00db84d5
3. Показать информацию о фильме
node cli.js --task=3 --id=602964b9737a927b00db84d5

node cli.js --task=5 --title="2001: A Space Odyssey"

node cli.js --task=6 --stars="Gary Lockwood"
7. Импорт фильмов с текстового файла (пример файла прилагается
